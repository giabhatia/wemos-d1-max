# wemos-d1-max
WeMos D1 Mini expansion board that allows stacking of multiple shields (relay/motor/sensor shields/etc..) onto a single D1 Mini

Key components visible:
U1 — WeMos D1 Mini (the main ESP8266 microcontroller)
U2, U3, U4 — WeMos Shield slots (labeled "WeMos Shield 1, 2, 3")
J3, J4 — Large pin headers on the right side for external connections, with labeled signals (TX, RX, D1–D8, 3.3V, 5V, GND, A0, etc.)
SW1 — A reset push button
D1 + R1 — An LED with a current-limiting resistor (status indicator)
J5, J6, J7 — Smaller connectors for inter-board communication between shields
